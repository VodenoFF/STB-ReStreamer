# STB-ReStreamer Documentation

This directory contains comprehensive documentation for the STB-ReStreamer project.

## Documentation Structure

The documentation is organized into three main sections:

### User Documentation (`/user`)
Documentation for end-users of the STB-ReStreamer application, including:
- Installation guides
- Configuration instructions
- Feature usage guides
- Troubleshooting information

### Developer Documentation (`/dev`)
Documentation for developers who want to understand or contribute to the STB-ReStreamer codebase:
- Architecture overview
- Code organization
- Development setup
- Contribution guidelines

### API Documentation (`/api`)
Documentation for the STB-ReStreamer API:
- REST API endpoints
- WebSocket events
- Data models
- Authentication

## Reading the Documentation

All documentation is written in Markdown format (.md files) which can be viewed directly on GitHub or using any Markdown viewer.

For the best reading experience:
1. Browse the documentation through the GitHub repository 
2. Use a Markdown editor or viewer (like VS Code with Markdown extensions)
3. Generate HTML documentation using a tool like MkDocs (instructions in the `/dev` section)

## Contributing to Documentation

If you find errors or want to improve the documentation:
1. Create a fork of the repository
2. Make your changes
3. Submit a pull request

Documentation contributions are always appreciated!
